# HEX Colors 
* HEX stands for Hexadecimal color code.

Structure:
- #RRGGBB

Each pair represents:

- RR → red
- GG → green
- BB → blue
```css
Example:

h1 {
  color: #ff0000;
}
#ff0000 = red
```
/* Examples: */

- #000000 → black
- #ffffff → white
- #ff0000 → red
- #00ff00 → green
- #0000ff → blue

/* Example: */
```css
body {
  background-color: #f4f4f4;
}
```
That is a light gray background.

# Short HEX

Sometimes HEX can be shortened.

Example:

- #ffffff → #fff
- #000000 → #000
- #ff0000 → #f00

Example:
```css
h1 {
  color: #fff;
}
```
