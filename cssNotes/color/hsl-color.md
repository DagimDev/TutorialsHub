# HSL Colors
HSL means:

- Hue
- Saturation
- Lightness

Structure:

- hsl(hue, saturation, lightness)

Example:

- color: hsl(0, 100%, 50%);

This equals red.

* Explanation:

# Hue

Color angle on the color wheel.

- 0 → red
- 120 → green
- 240 → blue

# Saturation

Intensity of color.

- 0% → gray
- 100% → full color
- Lightness

# Brightness.

- 0% → black
- 50% → normal
- 100% → white

Example:

- background-color: hsl(200, 80%, 60%);
# HSLA

Same as HSL but with opacity.

Example:

- background-color: hsla(200, 80%, 60%, 0.5);
