Transitions allow smooth changes between CSS property values over a specified duration.

1. transition-property css /* Specifies which property to animate */ transition-property: opacity;
2. transition-duration css /* Sets how long the transition takes */ transition-duration: 0.5s; /* or 500ms */
3. transition-timing-function css /* Controls acceleration curve */ transition-timing-function: ease-in-out; /* Other values: linear, ease, cubic-bezier(0.42, 0, 0.58, 1) */
4. transition-delay css /* Delays the transition start */ transition-delay: 0.2s;
5. Shorthand (All-in-One) css /* Order: property | duration | timing-function | delay */ transition: opacity 0.5s ease-in-out 0.2s;
6. Transitioning Multiple Properties css transition: opacity 0.3s ease, transform 0.5s cubic-bezier(0.25, 0.1, 0.25, 1);
